# Orbitum
中山大学软件学院2015春季学期软件设计课程课程设计——Orbitum

## 环境
+ win7 64位
+ vs2012
+ cocos2d-x 3.0

## 运行方式
+ 这个文件夹下面还缺少一个cocos2d的文件夹。这个文件夹是放cocos引擎源码的，很大，所以我没有上传。
+ 你配置好环境后新建一个项目``cocos new proj_name-l cpp -d . -p package_name``
+ 将新建项目中的cocos2d文件夹复制到我们项目的根目录下
+ 用vs打开proj.win32/Orbitum.sln
+ ctri+F5运行程序
